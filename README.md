Всем привет! 👋

Я — системный аналитик с опытом работы в банковской сферы. Умею анализировать пользовательские запросы, выявлять реальные потребности пользователей и переводить их в чёткую структурированную форму.
Интересуюсь системным анализом как областью на пересечении бизнес-процессов и пользовательских сценариев. Интересует создание понятных, удобных и практичных решений, приносящих ценность пользователям и бизнесу.

Ключевые навыки:
- сбор, анализ и формализация требований;
- управление и документирование требований;
- моделирование бизнес-процессов, данных, потоков данных и состояний;
- проектирование пользовательских интерфейсов;
- работа по Agile-методологиям;
- подготовка рабочей и проектной документации.

Инструменты и знания: Postman, SQL, базовые знания API и архитектуры систем, Miro, draw.io
